# 프로젝트 설정 및 데이터 백업 디렉토리 (Projects Directory)

이 디렉토리에는 2D 스프라이트 슬라이서 대시보드에서 생성된 프로젝트 설정 파일 및 백업 에셋들이 저장됩니다.

---

## 📂 주요 파일 구성 (Files)

* 📄 **dajung_pixel_settings.json**: `dajung` 캐릭터의 크로마키 값, 크롭 오프셋, 프레임 주기 등 이미지 슬라이싱 세부 픽셀 설정 정보입니다.
* 📄 **dajung_NeoMaid_2d_config.json**: 캐릭터의 상태(Idle, Walk, Run 등) 및 각 상태별 프롬프트 메타데이터 설정 정보입니다.
* 📦 **[6000.0.67f]dajung_NeoMaid_2d_x2fream.unitypackage**: 유니티 엔진 연동을 위한 패키지 파일입니다.
* 🎥 **_?_characterId_dajung_Neo.mp4**: 스프라이트 슬라이싱 테스트를 위한 소스 비디오 클립 파일입니다.
