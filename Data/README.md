# 캐릭터 설정 및 API 모듈 디렉토리 (Data Directory)

이 디렉토리에는 2D 스프라이트 슬라이서 대시보드에서 사용하는 캐릭터 기본 정보 설정 및 Google Gemini API 호출에 필요한 모듈이 저장됩니다.

---

## 📂 주요 파일 구성 (Files)

* 📄 **gemini.js**: Google Gemini API 호출을 위한 기본 구성 값(apiKey, modelName 등)을 포함하는 초기화 스크립트입니다. 보안을 위해 실제 API 키는 로컬 환경에서만 안전하게 기입하여 사용하시기 바랍니다.

## 1. ✅ 허용 사항 (Allowed Uses)
* **상업적 및 비상업적 이용 허용**: 본 리소스를 활용해 인디 게임, 2차 창작 콘텐츠를 제작하고 이를 수익화(상업적 판매 및 광고 수익 창출 등)할 수 있습니다.
* **수정 및 편집 허용**: 게임의 필요에 따라 도트 이미지를 수정하거나 색상을 변경하고, 메타데이터를 편집하는 행위를 허용합니다.
* **Commercial & Non-Commercial Use**: You are free to use these assets in indie games and fan-made content, and monetize them (e.g., commercial sales, ad revenue).
* **Modifications**: You are allowed to edit pixel art, change colors, or modify metadata as needed for your project.
